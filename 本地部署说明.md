# AI 评审系统 - 本地部署

## 快速启动

### 方式一:分别启动前后端(推荐)

**1. 启动后端**

```bash
cd backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python init_configs.py
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

**2. 启动前端**

```bash
cd frontend
npm install
npm run dev
```

访问: http://localhost:5173

### 方式二:使用启动脚本

```bash
# 终端1 - 启动后端
./start_local.sh

# 终端2 - 启动前端
./start_frontend.sh
```

## 配置说明

### 后端配置

编辑 `backend/.env` 文件:

```env
# OpenAI API 配置
OPENAI_API_KEY=your_api_key_here
OPENAI_API_BASE=https://api.openai.com/v1
OPENAI_MODEL=gpt-4
```

### 数据存储

- **数据库**: `backend/ai_review.db` (SQLite)
- **上传文件**: `backend/uploads/`

## 常见问题

### 1. 端口被占用

```bash
# 查看 8000 端口占用
lsof -i :8000
# 杀死进程
kill -9 <PID>
```

### 2. 前端连接后端失败

检查 `frontend/src/api/config.ts` 中的 API 地址:

```typescript
const API_BASE_URL = 'http://localhost:8000';
```

### 3. 数据库初始化失败

```bash
cd backend
rm ai_review.db
python init_configs.py
```

## 功能说明

- 产品评审文档上传
- AI 自动生成评审问题
- 评委人工提问
- AI 辅助评分
- 数据看板统计

## 技术栈

- 后端: FastAPI + SQLite + SQLAlchemy
- 前端: React + TypeScript + Tailwind CSS
- AI: OpenAI GPT-4
